╔══════════════════════════════════════════════════════════════════╗
║           BIZMASTERY — PRODUCTION DEPLOYMENT GUIDE              ║
╚══════════════════════════════════════════════════════════════════╝

TOTAL TIME: ~30 minutes from zero to live website
TOTAL COST: $10–12/year (domain only — everything else is FREE)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 1 — CREATE FREE ACCOUNTS (10 minutes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. GitHub: github.com/signup
   → Free. This stores your website code.

2. Vercel: vercel.com/signup
   → Click "Continue with GitHub" — links automatically.
   → This hosts your website for FREE.

3. Mailchimp: mailchimp.com
   → Free up to 500 contacts / 1,000 emails per month.
   → This stores everyone who downloads your guide.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 2 — GET YOUR API KEYS (15 minutes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

A) MAILCHIMP API KEY:
   1. Login → click your name bottom-left → Account & Billing
   2. Extras → API Keys → Create A Key
   3. Copy it. Looks like: abc1234567890xyz-us21
   4. The "us21" at the end = your MAILCHIMP_SERVER value

B) MAILCHIMP LIST ID:
   1. In Mailchimp → Audience → All Contacts
   2. Manage Audience → Settings
   3. Copy the "Audience ID" (8-character code)

C) GMAIL APP PASSWORD:
   1. Go to: myaccount.google.com
   2. Security → 2-Step Verification → turn ON (required first)
   3. Search "App Passwords" in the search bar
   4. Select app: Mail | Select device: Other → type "BizMastery"
   5. Copy the 16-character password Google generates
   NOTE: This is NOT your regular Gmail password

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 3 — SET UP THE CODE (5 minutes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Download the BizMastery project folder from Claude
2. IMPORTANT: Open public/index.html in a text editor
   - Find the function: handleEmailSubmit()
   - Replace its entire body with the code from: public/api-patch.js
   - Save the file

3. Create a file named ".env" (copy from .env.example):
   MAILCHIMP_API_KEY=paste_your_key_here
   MAILCHIMP_LIST_ID=paste_your_list_id_here
   MAILCHIMP_SERVER=us21
   EMAIL_USER=yourgmail@gmail.com
   EMAIL_PASS=paste_16_char_app_password_here

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 4 — PUSH TO GITHUB (5 minutes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Option A (with VS Code — easiest):
1. Open VS Code → Open Folder → select bizmastery folder
2. Click Source Control icon (left sidebar)
3. Click "Publish to GitHub" → name it "bizmastery" → public
4. Done — code is on GitHub

Option B (command line):
   Open terminal in the bizmastery folder, then run:

   git init
   git add .
   git commit -m "BizMastery launch"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/bizmastery.git
   git push -u origin main

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 5 — DEPLOY ON VERCEL (5 minutes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Go to: vercel.com/dashboard
2. Click "New Project"
3. Import from GitHub → select "bizmastery"
4. Before clicking Deploy → click "Environment Variables"
5. Add each variable:

   Name                  Value
   ──────────────────    ────────────────────────────────
   MAILCHIMP_API_KEY     abc1234567890xyz-us21
   MAILCHIMP_LIST_ID     a1b2c3d4e5
   MAILCHIMP_SERVER      us21
   EMAIL_USER            yourgmail@gmail.com
   EMAIL_PASS            abcd efgh ijkl mnop

6. Click DEPLOY
7. Wait ~60 seconds
8. Your site is LIVE at: https://bizmastery.vercel.app

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 6 — BUY AND CONNECT YOUR DOMAIN
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

BEST DOMAIN OPTIONS FOR YOUR SITE:
  → bizmastery.com         (check availability first)
  → mybizmastery.com
  → bizmastery.co
  → bizmasteryhq.com
  → businessmastery.co

WHERE TO BUY:
  Namecheap.com — best price (~$10/yr, includes free WhoisGuard privacy)
  GoDaddy.com   — well known (~$12/yr)

HOW TO CONNECT TO VERCEL:
  1. Buy domain on Namecheap
  2. Go to Vercel → your project → Settings → Domains
  3. Type your domain → Add
  4. Vercel shows you 2 DNS records to add
  5. Go to Namecheap → Domain List → Manage → Advanced DNS
  6. Add the records Vercel gives you (usually 2 CNAME records)
  7. Wait 5–30 minutes → your domain is live!

HOW THE EMAIL + PDF SYSTEM WORKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

User enters email →
  Server adds them to Mailchimp audience (tagged: bizmastery-guide) →
  Server generates the full PDF guide in memory using PDFKit →
  Server emails PDF as attachment via Gmail →
  User gets branded email with PDF in ~30 seconds

You see every subscriber in your Mailchimp dashboard.
You can email them future updates, offers, or content.
